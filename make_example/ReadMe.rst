====================
Пример Makefile
====================

Описание
--------
Этот Makefile демонстрирует автоматизацию процесса сборки и запуска C++ программы, манипуляции с директориями внутри make, и содержит описание корня системы Linux

Основные цели
-------------

1. ``my_dir``: Создает директорию ``my_dir``.
2. ``move_source``: Перемещает ``main.cpp`` в ``my_dir``.
3. ``compile``: Компилирует ``main.cpp`` в исполняемый файл.
4. ``run``: Запускает скомпилированную программу.
5. ``show_directories``: Выводит описание основных директорий Linux.
6. ``all``: Выполняет все вышеперечисленные цели последовательно.
